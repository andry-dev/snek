# snek
*a simple snek*

## About
*snek* is a *simple*(?) implementation of Snake that runs on a terminal, written in C. University project for the Computer Architectures Lab course.

## Build instructions
Run these commands:

	mkdir build
	cd build
	cmake ..
	cmake --build .
